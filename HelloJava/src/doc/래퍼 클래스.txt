래퍼 클래스.txt

기본형과 문자열 간의 변환방법

기본형 -> 문자열
문자열.valueOf(기본타입)
String.valueOf(기본타입)

ex: String.valueof(300);

문자열 -> 기본형
기본타입 클래스.parse타입(문자열)

ex: Integer.parseInt("300")

프로그래밍에서 반드시 알고 있어야 하는 아주 중요한 내용

Boolean, Byte와 같이 기본형 타입의 이름의 첫 글자가 대문자인 것은
래퍼 클래스이다. 기본형 값을 감싸고 있는 클래스라는 뜻에서 붙여진 이름으로
기본형을 클래스로 표현한 것이다.


*기본형 값을 래퍼 클래스의 객체로 자동 변환해주는 것을 오토박싱(autoboxing)이라 하고
반대로 변환하는 것을 언박싱(unboxing)이라고 한다
언박싱: Integer 객체를 int 기본형으로 자동 변환하는 과정.
오토박싱: int 기본형 값을 Integer 객체로 자동 변환하는 과정.

이와 같은 자동 변환 기능은 자바 컴파일러가 코드를 컴파일할 때 자동으로 처리해줍니다.
이를 통해 기본형과 객체형을 더욱 쉽게 사용할 수 있습니다.
